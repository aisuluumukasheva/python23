git init - создает git репозиторий из текущей папки
git add test.txt (git add.)- добавляет файлы и папки для отслеживания в index (эту коанду пишем каждый раз перед коммандой commit)
git commit -m 'сообщение' - создвет новую версию, если были изменения в отслеживающихся файлах

git log - показывает все коммиты
git status - показывает состояние файлов

git checkout ветка_или_id_коммита - переход на другую ветку или коммит
