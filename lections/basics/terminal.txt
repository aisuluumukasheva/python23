ls - показывает все файлы и папки внутри текущей директории
ls test - показывает все файлы и папки внутри папки test
ls -a - показывает также все скрытые файлы и папки
ls - l - показывает более подробную о файлах и папках

cd - переходит в домашнюю папку
cd ~ - переходит в домашнюю папку
cd test - переходит в папку test, если такая папка существует в текущей директории
cd. - переходит в текущую папку (остается в ней же)
cd.. - переходит в родительскую папку
cd../.. -  переходит на две папки назад

mkdir test - создает папку test в текущей директории
mkdir test/test2 - создает папку  test2 в папке  test, если папка test существует
mkdir test1 test2 - создает папки test1 test2 в текущей директории

touch test.txt - создать файл test.txt в текущей директории
touch test{1,2,3}.txt - создает файлы test1.txt и test2.txt

rm test - удаляет файл test, только при условии, если это не папка
rm-rf test - удаляет файл или папку со всем содержимым безвозвратно

mv  test/txt test/ - перемещает файл  test.txt в папку test
mv test.txt test2.txt - переименовываетфайл test.txt  в  test2.txt
mv test/ test2/ - переименовывает test в test2

history -  показывает все команды, которые выполнены в данном терминале
history 10 - показывает последние 10 команд
history | grep hello -  показывает все команды с использованием слова hello

pwd - показывает абсолютный путь (начиная с корневой папки/) путь до текущей директории
(print working directory)

cat test.text - показывает содержимое файла
nano test.txt - редактор кода, открывает файл test.txt в редакторе кода nano
code test - открывает папку или файл в редакторе кода VSCode

sudo - (super user do права суперюзера) 

(apt -  системный установщик пакетов)
(snap - установщик пакетов (чаще всего с каких-либо приложений))
